# Project Overview - Frontend

## Purpose
This is the **Epic.ai Frontend** application - a Next.js-based web application for managing LiveKit voice AI agents. The application provides:
- Agent creation and management (3-step wizard)
- Phone number provisioning and assignment
- Call history and analytics
- Dashboard with real-time statistics
- User settings and profile management

## Project Context
This is part of a larger LiveKit voice AI system with:
- **Frontend**: Next.js 15.5.6 application (this project)
- **Backend**: FastAPI/Python backend for API services
- **Database**: PostgreSQL via Prisma ORM
- **Authentication**: NextAuth v5 (beta.29)

## Key Features
- Multi-tenant voice AI agent platform
- Real-time dashboard with statistics
- Agent wizard for easy configuration
- Phone number management
- Call history with filtering
- Analytics and visualizations
- User profile and settings
